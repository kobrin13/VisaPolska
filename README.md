# About plwiza.user.js

Automatic form filling bot/userscript for [by.]e-konsulat.gov.pl in Chrome and Firefox

http://userscripts.org/scripts/show/178806

# Browsers Supported

* Chrome (v0.8+)
* Firefox + Scriptish or Greasemonkey plugin (v0.7+)

В картинках
===========
* Заполняем, проверяем и сохраняем в Excel данные для формы.

![excel](http://dela.by/ftp/plwiza/plwiza-excel.png)

* Заходим на сайт, конфигурируем скрипт (копируя E1:F6 в область настроек) или пользуемся настройками по-умолчанию.

![cfg](http://dela.by/ftp/plwiza/plwiza-cfg2.png)

* Попадая на форму заполнения, вставляем данные из Excel (столбцы A:D целиком) или пользуемся демонстрационным заполнением по-умолчанию.

![form](http://dela.by/ftp/plwiza/plwiza-form2.png)

О заполнении заявок на получение визы
=====================================

Программа-скрипт создавалась не для получения коммерческой прибыли. Тупое скоростное заполнение полей ввода html формы -- это просто не наш методЪ.

Там, где процесс работы не прозрачный, крутятся большие деньги, жди коррупцию. Моё неавторитетно-техническое мнение, что все проблемы с заполнением из-за коррупции (от админов белтелекома и консульства, до их нетехнических работников)...

Самый простой и белый механизм: публичный список людей, подавших в любое удобное время заявки. В дату Х проводится черта. Те, кто выше -- на выполнение, те кто ниже -- будут в следующий раз в первых рядах.

Так будет видно кто и как что подаёт.
